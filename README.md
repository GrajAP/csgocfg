# CS:GO Personal Autoexec

Personal CS:GO config I keep for training and game consistency.

## What is inside
- Keybindings for movement, inventory, and utility actions
- Mouse/aim feel adjustments and performance-related console settings
- Quick aliases for common binds and maps
- Network and radar HUD tuning variables

## Files
- `autoexec` — full config file with command list (`exec autoexec` from `config.cfg`/autoexec flow)

## How I use it
1. Copy `autoexec` to CS:GO cfg folder.
2. Add `exec autoexec` to startup config.
3. Start game and verify binds with `binds`/`host_writeconfig` when needed.

## Why this repo exists on GitHub
It shows practical console scripting / configuration work and how I document setup notes in a long-term config file.
